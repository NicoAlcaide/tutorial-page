1. ¿Qué es Angular?

Angular es un framework para desarrollar aplicaciones web con una arquitectura basada en componentes creada con TypeScript. Ofrece varias funciones y bibliotecas, como el enrutamiento y la comunicación cliente-servidor, que pueden ayudar a acelerar el desarrollo del front-end. Además, Angular proporciona a los desarrolladores herramientas para crear proyectos de cualquier tamaño, desde aplicaciones pequeñas hasta aplicaciones de nivel empresarial.

Angular es un framework front-end de JavaScript de código abierto desarrollado y administrado por el equipo Angular de Google. Se utiliza ampliamente para crear aplicaciones web y móviles escalables y de alto rendimiento mediante HTML, CSS y TypeScript.
